## Installation

Use the package manager [npm](https://npmjs.com) to install Zen Client's dependencies.
```bash
git clone https://github.com/AmnesiaLabs/zen-client
cd zen-client
npm i
```

## Running for development

```bash
npm run electron:serve
```

## Building

```bash
npm run electron:build
```
